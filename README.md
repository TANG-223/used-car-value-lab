# Used Car Value Lab - Upgraded Static Site

This package is a lightweight upgrade of the existing `used-car-value-lab` project.

## What's included
- `index.html` — complete page structure
- `style.css` — layout and visual design
- `script.js` — bilingual UI, estimator logic, presets, import/export JSON

## Main upgrades
- bilingual EN / 中文 toggle
- clearer project homepage and positioning
- estimator with presets
- reference range + explanation text
- contribution breakdown table
- educational sections: How It Works / Learn the Market / About

## Deployment
1. Replace the current site files in your GitHub Pages repo with these files.
2. Commit and push to GitHub.
3. GitHub Pages will publish automatically.

## Notes
- This remains a static educational demo site.
- No online model training is performed in the browser.
- Demo coefficients are illustrative and can be replaced with final coefficients from the student's notebook.
